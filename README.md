# Block Share App v2.0

**Citizens, not Consumers** — A community sharing platform for neighbourhood-level resource pooling.

## Overview

Block Share is a React Native mobile application that enables neighbours to share items, participate in food collectives, and build community through democratic governance. This v2.0 implementation includes the Host-centric operator model and simplified UX design.

## Features

### 4-Tab Simplified Structure

| Tab | Color | Purpose |
|-----|-------|---------|
| **Home** | Blue | Impact dashboard, quick actions, activity feed |
| **Stuff** | Green | Browse, borrow, and share items in pools |
| **Food** | Red | Food collective subscriptions (Beef Community Collective) |
| **Me** | Orange | Profile, credits, settings, Host tools |

### Key Features

- **Impact Dashboard**: Track money saved and "hours of life reclaimed"
- - **Category Grid**: 16 categories with large, easy-to-tap icons
  - - **Beef Community Collective**: Premium grass-fed beef at $14/kg flat pricing
    - - **Host Tools**: Training, announcements, resident management
      - - **Ranked-Choice Voting**: Fair decisions on pool priorities
       
        - ### Host-Centric Operator Model
       
        - Block Share operates through volunteer **Block Hosts** who organize and support sharing in their building/block.
       
        - ## Tech Stack
       
        - - **Framework**: React Native + Expo SDK 52
          - - **Navigation**: Expo Router v4
            - - **Language**: TypeScript
              - - **Icons**: Lucide React Native
               
                - ## Getting Started
               
                - ```bash
                  # Clone the repository
                  git clone https://github.com/TheDecentralist/block-share-app-v2.git
                  cd block-share-app-v2

                  # Install dependencies
                  npm install

                  # Start the development server
                  npm start
                  ```
